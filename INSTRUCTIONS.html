‚öôÔ∏è How to Customize Tips & Durations

You can easily change the prices and how long an effect lasts. Open main.py and find the self.fixed_tips section (around line 30):

self.fixed_tips = {
    33:  ('Sparkles', RainSparkleFilter(), 10),  # 33 = Tokens | 10 = Seconds
    99:  ('Big Eyes', BigEyeFilter(), 20),      # 99 = Tokens | 20 = Seconds
    200: ('Cyber Mask', FaceMask3D(), 30)       # 200 = Tokens | 30 = Seconds
}

To change a price: Change the number on the left (e.g., change 33 to 50).

To change duration: Change the last number in the brackets (e.g., change 10 to 60 for a full minute).

Note: After making changes, save the file and restart the script for the new settings to take effect.

üì∫ Quality Modes
  • 1080p (60 FPS): Recommended. Smooth performance and best for most internet connections.
  • 4K (30 FPS): Ultra-high detail. Requires a 4K camera and a powerful PC.
üé• Output Modes
  • Virtual Camera (vcam): The app acts as a webcam. In OBS, add a "Video Capture Device" and select "Unity Video" or "Python VCam".
  • Window Mode: Opens a window on your desktop. In OBS, add a "Window Capture" and select "AR_STREAM_WINDOW".
Pro Tip: Use Virtual Camera if you want to use the filters in other apps like Discord or Zoom. Use Window Mode if you want the lowest possible latency in OBS.
üçé macOS Users: The first time you run the script, macOS will ask for permission to allow "Terminal" (or your Code Editor) to access the Camera and Microphone. Click Allow for the filters to work.
